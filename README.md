# HabitFlow 🌱

> Tu tracker semanal de hábitos — construye la mejor versión de ti mismo.

## ✨ Funcionalidades

- ✅ Tabla semanal (Lun–Dom) × hábitos con checkboxes interactivos
- 💾 Persistencia automática en `localStorage`
- 📊 Score semanal con barra de progreso y anillo circular animado
- 🧠 Análisis qualitativo con insights personalizados
- 📅 Historial de semanas anteriores
- ➕ Agregar / editar / eliminar hábitos con selector de emojis
- 🎉 Confetti y micro-animaciones al completar hábitos
- 🌙 Dark mode premium con gradientes índigo/violeta
- 📱 Completamente responsive
